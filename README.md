### Usage

```swift
HUDLoading()
HUD(.loading("Loading..."))
HUD(.success("Success"))
HUD(.error("Operation Error"))
HUD(.warning("watch Out!"))
HUD(.content("Hi, FGHUD"))
HUD(.toast("Hi, FGHUD"))
HUDHide()
```

### Install
```
pod repo update
pod 'FGHUD', '~> 3.0.3'
import FGHUD
```
